# Flappy Bird Game

Super simple Flappy Bird clone made with HTML, CSS, and JavaScript. Runs right in your browser—no setup needed!

### Quick Play
- Click or tap the screen (or press Space) to make the bird flap
- Dodge the pipes and rack up points
- High score saves automatically

### How to Run
1. Grab the three files: `index.html`, `style.css`, `script.js`
2. Put them in the same folder
3. Open `index.html` in any browser
4. Play!

https://pukardahalx.github.io/flappy-bird-game/

It's chill and easy to control—perfect for a quick break. Enjoy!
